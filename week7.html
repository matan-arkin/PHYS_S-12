Week 7
Electronic Input Devices

Part I - Microphone

With the final project in mind, I was looking to get some experience with microphones. Using the SPW2430 included in the kit, I wired ground (GND) and 3.3v (Vin) to the Metro, then wired the DC output to pin A0 like so:



Then I prepared for a calibration test. I set up my bluetooth speaker on one end of the table and the SPW2430 on the other, with my iPhone bordering the mic to its right, careful to place them at equal distances from the speaker. I installed an app on my iPhone that measured decibals (dB), and left it running. Finally, I found a consistent source of sound in the form of a YouTube video playing, from my laptop and through the speaker, a 432Hz soundwave for 8 hours (yes, it sounded awful).



Part I - Calibration

For each measurement, I adjusted the volume from the laptop until I saw the dB value I wanted. I increased by 5dB for each step, though my speaker had a hard time generating exactly 50 dB, so I omitted that value. The code I used was based off of the microphone page on the course website. I added a higher, 12-bit resolution because I wasn't able to differentiate between values on the lower end of the dB scale. I was also seeing too many values to clearly interpret, even using Serial Plotter. I didn't want to just increase the sampling window because that would give me the most extreme values in each, large, window. Instead, I used a for loop that repeated the 50mS window 10 times and averaged the PeakToPeak values. The code is available for download here. I entered the values into Excel and graphed a curve.



The results turned out non-linear with 45dB corresponding to a measured value of 16 and 100dB corresponding to a measured value of 70. I then recalled that dB was a logarithmic scale that could be converted to regular magnitude. Using the formula MAG=10^(DB/20) from this site, I added an excel column for magnitude and tried graphing it to see if it would turn out linear. The answer: sort of!







Part II - Scale

For my sensor fabrication, I wanted to create a scale. We have a scale in the house, but it's for heavy objects. I was hoping to build a sensor for finer measurements. I started off with the two copper plates from the kit and a layer of foam I found lying around. I wired up the tx copper plate directly to A1 and the rx plate to A3 through a pair of 1MOhm resistors to ground and 3.3V. For code, I used the template rx/tx code from the course website and adjusted N_samples to 500 so I would have enough time to read the values coming in. The code is available for download here. Part II - Calibration

Like in part I, I used known values, this time of the weight of water, to interpret the measurements from my scale. I used an empty beaker and added 100 ml of water, which should weigh roughly 100 grams, at each step. I recorded the measurement from Serial Plotter to the same Excel as before, in a different sheet. That Excel is available for download here.


Here too, the results do not seem to be linear. Perhaps as more force is applied to the foam, it becomes increasingly difficult to compress it further!
