Purpose

This is a SCUBA mask from 40 years ago.



This is a "modern" mask.



Introducing Ncompass. A SCUBA mask for 2021 that empowers divers to "see" the sounds of the ocean.

As every SCUBA diver knows, while sounds can be heard during dives, it is impossible to tell where a sound is coming from. Ncompass will detect the source of sounds underwater in real time and light up that direction so you know where you are and what's going on around you.



Why Ncompass?

  • Navigation
  • Safety
  • Affordability

Navigation

Navigate underwater intuitively using sounds to identify your surroundings. Engine noises to your left? That's probably the boat channel. Stay away. Dolphin whistles to your right: might want to check that out.

Safety

Identify threats in real time. Where is that banging noise coming from?

Affordability

Pick up Ncompass for only $29.99 - coming soon. You know best which mask is right for you, so we designed Ncompass to work with your mask.

How It Works

Sound Waves Underwater, Hardware

The reason we cannot perceive directionality of sound underwater has to do with the speed of sound. Underwater, sound travels 4 times faster than in air. The way our brains normally deduce directionality is by comparing the miniscule delay between one ear's detection of a sound and the other's. Together with slightly different amplitudes ("loudness"), our brains process all this information and perceive a direciton. Underwater, this process breaks down because the time between sound hitting one ear and sound hitting the other is 4 times smaller. Ncompass uses this same mechanism, replacing ears for a pair of microphones and the brain for a small microprocessor. Ncompass executes fast enough to contend with the underwater speed of sound, then updates LED lights accordingly to intuitively and instantaneously communicate the result to the diver.


Software


#include <Adafruit_NeoPixel.h>

#define LEDSTRP     6
#define MICPINR     A0
#define MICPINL     A5
const int NUMPIXELS = 10;     // 10 LEDs on LED strip
const int SAMPLEWINDOW = 100; // Sample window width in mS
const int PEAK_THRESH = 30;   // loud enough to be a clap

Adafruit_NeoPixel strip(NUMPIXELS, LEDSTRP, NEO_GRB + NEO_KHZ800);

void setup() {
  Serial.begin(250000);
  strip.begin();            // INITIALIZE NeoPixel strip object
  strip.setBrightness(100); // (max = 255)
  strip.clear();            // Set all pixel colors to 'off'
  strip.show();             // Turn OFF all pixels ASAP
}

void loop() {
  int signal_max_left = 0;
  int signal_min_left = 1024;
  int signal_max_right = 0;
  int signal_min_right = 1024;
  unsigned long start_millis = millis(); // Start of sample window

  // sample p2p left and right for 1 SAMPLEWINDOW
  while (millis() - start_millis < SAMPLEWINDOW) {
    int sample_left = analogRead(MICPINL);
    int sample_right = analogRead(MICPINR);
    if (sample_left < 1024 && sample_right < 1024) { // toss out spurious readings
      if (sample_left > signal_max_left)
        signal_max_left = sample_left;
      else if (sample_left < signal_min_left)
        signal_min_left = sample_left;
      if (sample_right > signal_max_right)
        signal_max_right = sample_right;
      else if (sample_right < signal_min_right)
        signal_min_right = sample_right;
    }
  }
  // max - min = peak-peak amplitude
  int p2p_left = signal_max_left - signal_min_left;
  int p2p_right = signal_max_right - signal_min_right;

  // if loud enough, change side
  if (p2p_left > PEAK_THRESH || p2p_right > PEAK_THRESH) {
    strip.clear();
    // change to the louder side 
    if (p2p_left + 2 > p2p_right) // adjust diff in p2p sensitivities
      strip.fill(strip.Color(150, 0, 0), 0, 5);
    else
      strip.fill(strip.Color(150, 0, 0), 5, 5);
    delay(500);
    strip.show();
  }
}
